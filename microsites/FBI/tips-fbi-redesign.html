The current tips.fbi.gov is a fully JavaScript-rendered Angular application with no server-side fallback. The form shell contains a single <app-root></app-root> tag — if JS fails or loads slowly, citizens see a blank page. It loads Google Maps, Google Tag Manager, and reCAPTCHA before showing anything. The multi-stage crime-type routing forces citizens to self-categorize before submitting — a burden the FBI's own intake process handles internally anyway.

100% client-side JavaScript rendering Architecture
Zero server-rendered content. No-JS fallback absent. Fails WCAG 4.1.1. Screen readers receive an empty document until Angular bootstraps.
Google Maps required for address entry WCAG 1.3.3
Address field depends on the Maps JavaScript API. If the API is slow or blocked, citizens cannot enter a location at all. Sensory characteristic dependency.
Crime-type categorization forced before submission UX / Routing
Citizens must navigate a decision tree to find the right category before they can describe what they saw. FBI's own documentation states every tip is reviewed by two analysts regardless of category — the burden is unnecessary and creates drop-off.
reCAPTCHA v3 loaded on page entry WCAG 1.1.1
Invisible reCAPTCHA tracks users before they've interacted with the form. No accessible alternative provided. Cognitive and privacy barrier for users with assistive technology.
Google Tag Manager / Analytics before consent Privacy / Trust
GTM fires before the user has submitted anything. On a form citizens use to report crimes — often involving personal risk — pre-submission analytics tracking is a trust violation regardless of legality.
Generic system font stack Design
font-family: sans-serif, Arial, Helvetica. No type system, no visual hierarchy investment. A form asking for sensitive civic information should communicate institutional reliability through design.
No visible alternate paths on landing WCAG 2.4.1
IC3.gov, NCMEC, field offices, and disaster fraud — all faster paths for specific crime types — are not surfaced before or alongside the primary form. Citizens who don't know to look elsewhere are routed through the wrong channel.
