About this data

El Paso County is served by two separate agencies with very different data systems. The agency selector above switches what you see below.

CSPD open data

City of Colorado Springs only — ~506k residents inside city limits. Real-time API, 511K+ records, Jan 2020–present, updated daily. Full zip code breakdown and monthly trend available.

EPSO (Sheriff)

Unincorporated El Paso County — ~2,130 sq mi including Black Forest, Falcon, Peyton, Security-Widefield, rural east county. Annual report only — no real-time API, no zip-level data, no monthly breakdown publicly available.

Coroner (EPCCO)

All of El Paso County — city + unincorporated. 2024 Annual Report. 6,310 cases investigated. 2025 data expected Q3 2026.

Fountain PD, Manitou Springs PD, Monument PD, and Colorado State Patrol publish no real-time crime data APIs and are not included.
EPSO real-time data does not exist publicly. The Sheriff's Office does not publish an incident-level API or open data portal. What you see below is the 2025 Annual Report summary and 2024 coroner data — the only publicly available EPSO crime statistics. Their official crime map page directs users to LexisNexis, which shows CSPD data only. See the Findings tab for the full disclosure gap analysis.

CriticalTransparencyData

EPSO directs the public to communitycrimemap.com. That platform shows only CSPD data (Agency ID 532). EPSO patrols 2,130 sq mi with 153,000+ annual calls for service — none publicly accessible in real time.

  • CRS 24-72-203CORA — right to inspect public records including crime incident data
  • CRS 24-72-301CCJRA — criminal justice records open for inspection unless specifically excepted

Required action
EPSO must publish a real-time incident data API — or explicitly disclose that no EPSO data is publicly available rather than redirecting to a CSPD-only tool.

CriticalDataTransparency

EPSO's official tool showed 272 records in 28 days. CSPD open data shows ~3,585 equivalent incidents. The gap is structural: LexisNexis caps at 500 records, shows only 9 of 28+ crime categories, and this limitation is not disclosed.

9. Three municipalities — no accessible public crime data
MediumTransparency

Fountain (~34k residents), Manitou Springs (~6k), and Monument (~10k) have their own police departments but no real-time crime data APIs. ~50,000 El Paso County residents have zero access to real-time crime data for their city.

  • CRS 24-72-201 et seq.CORA — all public bodies including municipal police departments must make public records available
  • CRS 29-21-101Model transparency act — encourages proactive digital publication by local governments
Recommended action
El Paso County should pursue a regional open data agreement using the existing Tyler Data & Insights platform at policedata.coloradosprings.gov.
10. Coroner 2025 data not published — one-year public lag
MediumData

Most recent coroner data covers 2024 (released Sept 2025). 2025 homicide, suicide, and overdose statistics will not be public until ~Q3 2026. No quarterly release exists.

  • CRS 30-10-601Coroner records are public records — no proactive quarterly publication obligation currently exists, creating a gap between the access right and practical public benefit
Recommended action
EPCCO should publish quarterly mortality trend data (cause and manner totals only, no PII) to reduce the 9-12 month public lag.
